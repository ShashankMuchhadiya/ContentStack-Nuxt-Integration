# ContentStack Integration Documentation

Welcome to the documentation for the Nuxt ContentStack Integration project. This is a modern, dark-mode-first Nuxt.js application integrated with ContentStack CMS.

## What is this?

This project provides a clean, Apple-inspired UI with seamless content management capabilities, multi-language support, and comprehensive component documentation.

## Features

- **Modern Dark Mode UI**: Beautiful, minimalist dark theme inspired by Apple.com
- **ContentStack CMS Integration**: Full integration with ContentStack for content management
- **Server-Side Rendering (SSR)**: Optimized for SEO and performance
- **Internationalization (i18n)**: Multi-language support with 4 languages
- **TypeScript Support**: Full TypeScript support for type safety
- **Component Documentation**: Storybook integration for component development
- **State Management**: Pinia store for efficient data caching
- **SEO Optimization**: Built-in SEO support with meta tags

## Quick Start

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Set up environment variables**
   Create a `.env` file with your ContentStack credentials

3. **Run the development server**
   ```bash
   npm run dev
   ```
