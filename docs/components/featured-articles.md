# FeaturedArticles Component

Featured articles listing component.

## Usage

```vue
<template>
  <FeaturedArticles :articles="articlesData" />
</template>
```

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `articles` | `FeaturedArticles` | Yes | Articles data from ContentStack |

## Features

- Article listing
- Image thumbnails
- Publication dates
- Article links
