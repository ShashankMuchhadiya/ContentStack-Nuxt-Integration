# Footer Component

Footer component with dynamic sections from ContentStack.

## Usage

```vue
<template>
  <Footer />
</template>
```

The Footer component automatically fetches menu data from ContentStack and displays it.

## Features

- Multi-column layout
- Dynamic sections from ContentStack
- Rich text support
- Responsive design

## Props

The Footer component doesn't require any props. It automatically:
- Fetches menu data from ContentStack
- Detects the current language
- Updates on language changes
