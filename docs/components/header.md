# Header Component

Navigation header component with language switcher and dynamic navigation from ContentStack.

## Usage

```vue
<template>
  <Header />
</template>
```

The Header component automatically fetches navigation data from ContentStack and displays it.

## Features

- Sticky navigation with glassmorphism effect
- Responsive mobile menu
- Language switcher integration
- Smooth hover animations
- Loading states
- Dynamic navigation from ContentStack

## Props

The Header component doesn't require any props. It automatically:
- Fetches navigation data from ContentStack
- Detects the current language
- Updates on language changes

## Styling

The header uses:
- Dark mode theme
- Glassmorphism effects
- Smooth transitions
- Responsive breakpoints
