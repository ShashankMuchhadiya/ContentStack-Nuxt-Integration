# LanguageSwitcher Component

Language selector dropdown component.

## Usage

```vue
<template>
  <LanguageSwitcher />
</template>
```

## Features

- Dropdown language selector
- Automatic URL updates
- Locale persistence
- Smooth animations

## Supported Languages

- English (en)
- Spanish (es)
- French (fr)
- German (de)

The component automatically handles:
- URL updates when language changes
- Cookie persistence
- Content refresh for new language
