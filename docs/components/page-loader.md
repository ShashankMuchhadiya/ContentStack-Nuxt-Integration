# PageLoader Component

Loading spinner component that displays while content is being fetched.

## Usage

```vue
<template>
  <PageLoader :message="'Loading content...'" />
</template>
```

## Props

| Prop | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `message` | `string` | No | `"Loading..."` | Loading message to display |

## Features

- Customizable loading message
- Smooth animations
- Dark mode styling
