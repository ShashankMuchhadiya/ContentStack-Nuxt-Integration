# API Reference

This section contains detailed API documentation for utilities and SDK functions.

## ContentStack SDK

The ContentStack SDK utilities provide low-level access to ContentStack functionality.

### [ContentStack SDK](./contentstack-sdk.md)

Core SDK utilities for interacting with ContentStack.

## Utilities

### [Utilities](./utilities.md)

Helper functions for common tasks like link resolution, cache key generation, and locale mapping.

## Query Functions

- `fetchSingleEntry` - Fetch a single entry from ContentStack
- `fetchMultipleEntries` - Fetch multiple entries from ContentStack
- `fetchAllEntries` - Fetch all entries of a content type

## Error Handling

The project includes comprehensive error handling utilities for ContentStack API errors.
